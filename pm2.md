<h1>pm2日常使用</h1>
	<h3>⭕️ pm2是一个带有负载均衡功能的Node应用的进程管理器</h3>
	<p>安装: npm install -g pm2</p>
<strong>主要特性</strong>
<ul>
	<li>内建负载均衡(使用Node cluster 集群模块)</li>
	<li>后台运行</li>
	<li>0秒停机重载</li>
	<li>具有ubuntu和centos的启动脚本</li>
	<li>停止不稳定的进程(避免无限循环)</li>
	<li>控制台检测</li>
	<li>提供HTTP API</li>
	<li>远程控制和实时的接口api</li>
</ul>
